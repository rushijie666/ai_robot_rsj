1. Docker 进阶命令
查看正在运行的容器：

docker ps

查看所有容器：

docker ps -a

查看本地镜像：

docker images

保存容器为新镜像：

docker commit -m "install opencv and pybullet" -a "student" 容器ID my-ros2-env:v1.0

2. GitHub Pages 部署步骤
打开 GitHub 作业仓库
进入 Settings
找到 Pages
Source 选择 Deploy from a branch
Branch 选择 main
Folder 选择 root
保存后等待网页部署完成
