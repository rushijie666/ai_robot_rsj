PID 控制中 P、I、D 分别表示什么？
P 表示比例控制，根据当前误差进行调整。
I 表示积分控制，根据过去误差累积进行修正。
D 表示微分控制，根据误差变化速度进行提前调整。

避障逻辑的基本思路
机器人通过传感器检测前方距离。
如果前方没有障碍物，就继续前进；如果距离太近，就后退并转向，避免碰撞。
